Alinhamento de tabelas

O alinhamento de tabelas em HTML pode ser feito utilizando propriedades CSS como text-align para alinhar o conteúdo das células e vertical-align para alinhar verticalmente. E não só como também podemos utilizar a propriedade margin para adicionar espaçamento ao redor das células etc. Exemplo de alinhamento de tabelas:

Coluna 1 Coluna 2 Coluna 3
A1 B1 C1
A2 B2 C2
A3 B3 C3
