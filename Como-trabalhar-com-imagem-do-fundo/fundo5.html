Alinhamento vertical

Esta é uma página de exemplo para demonstrar o alinhamento vertical de elementos. Para o alinhamento vertical, é necessário que utilizemos as propridade de posicionamento realivo (position: relative) para o container e o posionamento aboluto (position: absolute) para o conteúdo. Quando utizamos a propriedade position: absolute, podemos controlar a posição do elemento dentro do container posicionado de forma relativa com as propriedades top, left, bottom e right e a propriedade transform: translate() que permite mover o elemento de forma mais flexível.
